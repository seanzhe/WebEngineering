Google Maps API

Overview

  • Documentation: https://developers.google.com/maps/documentation/javascript/tutorial?csw=1
  • Uses HTML5.
  • You can write a page that uses Google Maps and run it from your desktop (with Internet connection); no web server required.
  • Requirements:
    1. In your the JavaScript source, before the end of the closing </body> tag <script src="https://maps.googleapis.com/maps/api/js?key=YOUR_GOOGLE_MAPS_JS_API_KEY_HERE&callback=init" async defer />
    2. CSS: 
      	html { height: 100% }
	body { height: 100%; margin: 0px; padding: 0px }
	#map_canvas { height: 100% }
  • Tutorial: https://developers.google.com/maps/documentation/javascript/tutorial
  • Objects:
    • google.maps.Map - The map object (duh!)
    • google.maps.LatLng - An object that contains the latitude and longitude pair
    • google.maps.Marker - A marker
    • google.maps.InfoWindow - An info window
    • google.maps.Polyline - A linear overlay
    • google.maps.event - An event listener for Google Maps
  • Complete reference: https://developers.google.com/maps/documentation/javascript/reference?csw=1

Structure of a Google Maps API version 3 Page

<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="initial-scale=1.0, user-scalable=no" />

<!-- Better put the following style into a separate CSS file -->
<style>
  html { height: 100% }
  body { height: 100%; margin: 0px; padding: 0px }
  #map_canvas { height: 100% }
</style>
<script src="http://maps.google.com/maps/api/js?sensor=true"></script>
<script>
	function initialize() {
		latlng = new google.maps.LatLng(...);
		// "..." is stuff you have to fill in
		...
		...
		...
		myOptions = {...};
		map = new google.maps.Map(document.getElementById("map_canvas"), myOptions);
	}
</script>

</head>
<body onload="initialize()">
	<div id="map_canvas" style="width:100%; height:100%"></div>
</body>
</html>
