jQuery

Why Use a Good JavaScript Library?

  • Rapid development
  • Wrapper for JavaScript
  • Lightweight
  • Shorter code to write
  • Allows for developers to create abstractions (i.e., plugins)
  • A lot of browser incompatibility worries are alleviated
  • Find stuff and do something to it
  • Plethora of JavaScript libraries available

Why jQuery?

  • https://jquery.com/
  • Outstanding documentation
  • Lightweight (~19 KB for minified format)
  • Simple code and easy syntax
  • Many plugins and special effects
  • Ajax support
  • Cross-browser support
  • Plugins --a lot of cool stuff and features you need are already written (e.g., a popup calendar, scrolling news ticker)
  • Do not think that jQuery IS JavaScript
  • Legacy

Getting Started: Two Ways

  1. Download jQuery and include it in your HTML pages: <script src="jquery-3.5.1.min.js"></script>
  2. Use jQuery directly: <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>. Of course, you have to be connect to the Internet and have to depend on jQuery being up!

Launching Code on Document Ready

 $(document).ready(function(){
   // Your code here
 });
  • Checks and waits until the document object is ready to be manipulated (that is, everything including images are loaded).

Selectors

  • jQuery uses CSS-style selectors to find specific elements in the DOM
  • $('h1,h2,h3')
  • $('div#content p.blurb')
  • $("a")
  • $("div")
  • $('#myForm :radio:checked') - filter a form element
  • $('li:first') - find a specific element
  • Notice, you can use single quote or double quotes around the elements
  • jQuery selection methods all return jQuery collections --that is, an array. Examples:
    • $('div').length
    • $('div')[0]
    • $('div')[1]
    • $('div').size()
    • $('div').each(function() { console.log(this); });
  • Exercise: Using jQuery, implement a feature that highlights (in yellow) all the paragraphs in the document, and echo all the text inside of the paragraph tags to a console (e.g., in Firebug).

Syntax

  • $("#header") - Get the element with id="header"
  • $("h3") - Get all <h3> element
  • $("div#content .photo") - Get all element with class="photo" nested in the <div id="content">
  • $("ul li") - Get all <li> element nested in all <ul>
  • $("ul li:first") - Get only the first <li> element of the <ul>

Magic Selectors

  • :first

HTML Manipulation

  • $('div').html()

  • $('div').append('<p>Hi!</p>')
  • $('div').after('<p>Hi!</p>')

Attributes

  • $('img').attr('alt')
  • $('img').attr('alt', 'Logo')
  • $('img').attr({ src: 'logo.png', alt: 'Logo' });

  • $('input').val('First Name')

Changing the Class of Elements
